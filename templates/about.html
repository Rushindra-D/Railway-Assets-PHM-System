Technical Overview

Railway PHM System

An advanced Prognostics and Health Management platform utilizing high-frequency vibration signal analysis to ensure railway operational safety.

🎯

Primary Objective

To provide real-time monitoring and predictive insights for railway asset health, enabling proactive maintenance, minimizing unplanned downtime, and significantly reducing mechanical failure risks.

🔧 Key System Features

✔ Feature extraction from vibration signals
✔ Normalization using healthy baseline
✔ Anomaly detection via Isolation Forest
✔ Real-time Health Index computation
✔ Remaining Useful Life (RUL) estimation
